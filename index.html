Geological Observations

This map contains a number of geological observations, presented as Feature Services hosted on ArcGIS Online.

The addition of a WMS layer provided by the BGS allows the observations to be shown within their geological context. This WMS layer is available at scales of 1:100,000 and below.

The GetFeatureInfo capability of the WMS layer is exploited to construct popup windows that contain useful information.

Operational layers are listed below. The WMS layer can sometimes be less responsive than the Feature Services. Try panning the map or refreshing the page if problems occur.
